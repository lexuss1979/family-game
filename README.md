# Большой семейный сбор — микро-демо

Небольшой игровой прототип на Godot 4.7. Персонажи и полы уже используют подготовленные игровые ассеты; остальная временная графика пока рисуется кодом.

## Запуск

1. Откройте `project.godot` в Godot.
2. Нажмите `F5` или кнопку запуска проекта.

Для подключения второго компьютера и совместной работы используйте [пошаговую инструкцию](docs/setup-on-second-pc.md).

Управление:

- `WASD` или стрелки — движение;
- `E` или `Пробел` — взаимодействие.

Цель микро-демо: открыть дверь, найти пульт в спальне и отнести его к дивану в гостиной.

Дом больше окна игры: камера следует за папой. Папой управляет игрок, остальные семь персонажей самостоятельно чередуют ожидание и короткие прогулки по комнатам.

У каждого члена семьи есть собственный набор случайных реплик. Баблы появляются через разные интервалы, следуют за персонажами и скрываются автоматически.

Фоновая chiptune-музыка синтезируется в реальном времени без внешнего аудиофайла. Клавиша `M` включает и отключает музыку.

При запуске показывается анимированный стартовый экран с логотипом. Игра начинается после нажатия кнопки, `Enter` или `Пробела`.

## Полы

Шесть покрытий из файлов в `design` автоматически нарезаются в `assets/tiles/floors`. В демо одновременно показаны светлое дерево, зелёная кухонная плитка, ковёр и голубая плитка ванной. Узорная плитка прихожей и тёмное дерево также готовы для следующих комнат.

```powershell
& "C:\Tools\Godot\4.7\Godot_v4.7-stable_win64_console.exe" --headless --path . --script res://tools/extract_floor_tiles.gd
```

## Стены и окна

Стены собраны из технического атласа `assets/tiles/walls/wall_terrain.svg`: 16 вариантов соединения расположены в сетке 4×4 с ячейкой `64×64`. Индекс варианта является четырёхбитной маской соседей: слева — `1`, справа — `2`, сверху — `4`, снизу — `8`.

Окно хранится отдельно в `assets/tiles/walls/window_topdown.svg` и накладывается поверх прямого участка стены. Один и тот же ассет используется горизонтально или поворачивается на 90 градусов для вертикальной стены.

## Персонажи

Атласы всех персонажей находятся в `assets/characters`. Они автоматически извлекаются из исходного листа `design/personage.png` скриптом:

```powershell
& "C:\Tools\Godot\4.7\Godot_v4.7-stable_win64_console.exe" --headless --path . --script res://tools/extract_characters.gd
```

Каждый атлас содержит девять ячеек `160×160` в порядке: `idle`, `up 1/2`, `down 1/2`, `left 1/2`, `right 1/2`.
