# Большой семейный сбор — микро-демо

Небольшой игровой прототип на Godot 4.7. Временная графика рисуется кодом и позже будет заменена подготовленными тайлами и спрайтами.

## Запуск

1. Откройте `project.godot` в Godot.
2. Нажмите `F5` или кнопку запуска проекта.

Управление:

- `WASD` или стрелки — движение;
- `E` или `Пробел` — взаимодействие.
- `Tab` или кнопка в верхней панели — следующий персонаж.

Цель микро-демо: открыть дверь, найти пульт в спальне и отнести его к дивану в гостиной.

## Персонажи

Атласы всех персонажей находятся в `assets/characters`. Они автоматически извлекаются из исходного листа `design/personage.png` скриптом:

```powershell
& "C:\Tools\Godot\4.7\Godot_v4.7-stable_win64_console.exe" --headless --path . --script res://tools/extract_characters.gd
```

Каждый атлас содержит девять ячеек `160×160` в порядке: `idle`, `up 1/2`, `down 1/2`, `left 1/2`, `right 1/2`.
